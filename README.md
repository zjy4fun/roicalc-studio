# ROICalc Studio

A static-first micro-SaaS MVP for building branded ROI calculators that turn B2B website visitors into sales conversations.

## Why this can make money

The fastest monetization path is service revenue before subscriptions:

- **Starter service — $299:** one branded calculator, embed snippet, and conversion copy angle.
- **Growth package — $999:** calculator + landing page + lead capture integration + offer variants.
- **Productized SaaS later:** hosted calculators, analytics, CRM integrations, team workspaces, and white-label domains.

Ideal customers: B2B SaaS founders, agencies, fractional CMOs, and consultants that need a business-case asset for demos and proposals.

## MVP features

- Interactive ROI scenario engine.
- First-year gain, ROI percentage, payback period, new-revenue and gross-profit outputs.
- Brandable embeddable HTML widget generator.
- Conversion-focused landing page with pricing/positioning.
- Static deployment-ready Vite + React app.

## Development

```bash
npm install
npm run dev
npm test
npm run lint
npm run build
```

## Deployment

The app is configured for GitHub Pages with `base: './'` in `vite.config.ts`.
